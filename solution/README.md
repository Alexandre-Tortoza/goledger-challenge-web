# Manifesto

Por meio deste manifesto, declaro que, para a implementação principal deste teste técnico, não utilizarei ferramentas de geração de código baseadas em IA, como Claude, Codex ou similares.

Embora essas ferramentas possibilitem maior velocidade no desenvolvimento, o objetivo desta etapa é permitir uma avaliação clara das minhas habilidades individuais, incluindo raciocínio, tomada de decisão e domínio técnico.

Como complemento, disponibilizo uma implementação alternativa em uma branch separada, na qual faço uso de ferramentas de IA. Essa versão tem como objetivo demonstrar minha capacidade de utilizar essas ferramentas de forma eficiente e produtiva.

A proposta é oferecer uma visão comparativa entre as abordagens, considerando aspectos como:

- Qualidade do código
- Experiência do usuário (UX/UI)
- Tempo de desenvolvimento

Dessa forma, fica possível avaliar não apenas minha base técnica, mas também minha capacidade de adaptação e uso consciente de ferramentas modernas no processo de desenvolvimento.
